# Unit 1: Identity Statements - Substitution Mode Reference

This document shows all Unit 1 sentences in **Substitution Mode**, demonstrating how they sound when pronounced.

## Presentation Formats

The substitution system has two presentation formats:

### 1. Simplified Format (Original)
Shows phonetic spelling: `Eu sou u Danieu`

### 2. Dictionary-Style Format (Current in Unit 1 Lesson)
Shows professional respelling with stress marking: `EH-oo SOH oo dah-nee-EH-oo`
- **CAPITALS** indicate stressed syllables
- **Hyphens** separate syllables
- English-like letter combinations for pronunciation

**Note:** Both formats apply the same 6 pronunciation rules. Only the presentation style differs.

## Pattern 1.1 - Names (Eu sou [o/a] [name])

| Original | Annotated | Substituted (Simple) | Dictionary-Style |
|----------|-----------|----------------------|------------------|
| Eu sou o Daniel. | Eu sou o/u/ Daniel/u/. | Eu sou u Danieu. | EH-oo SOH oo dah-nee-EH-oo |
| Eu sou a Sofia. | Eu sou a Sofia. | Eu sou a Sofia. | EH-oo SOH ah so-FEE-ah |

**Key changes:**
- `o` → `u` → "oo" (article)
- `Daniel` final `l` → `u` → "EH-oo" (Rule 6: syllable-final L)

## Pattern 1.4 - Cities (Eu sou de [city])

| Original | Annotated | Substituted (Simple) | Dictionary-Style |
|----------|-----------|----------------------|------------------|
| Eu sou de Miami. | Eu sou de/dji/ Miami. | Eu sou dji Miami. | EH-oo SOH jee mee-AH-mee |
| Eu sou de São Paulo. | Eu sou de/dji/ São Paulo/u/. | Eu sou dji São Paulu. | EH-oo SOH jee SOW POW-loo |
| Eu sou de Nova York. | Eu sou de/dji/ Nova York/i/. | Eu sou dji Nova Yorki. | EH-oo SOH jee NOH-vah YORK-ee |

**Key changes:**
- `de` → `dji` → "jee" (Rule 3a: palatalization)
- Final `-o` → `u` → "oo" (Paulo → POW-loo)
- Final consonant → add `i` → "ee" (York → YORK-ee, Rule 4b: epenthetic i)
- **São** has tilde - no change needed (already nasal)

## Pronunciation Key

### Dictionary-Style Notation

**Format conventions:**
- **CAPITALS** = stressed syllable
- **Hyphens** = syllable breaks
- ***(nasal)*** = nasal vowel sound

### Transformation Rules Applied

| Rule | Portuguese | Simple Format | Dictionary-Style |
|------|------------|---------------|------------------|
| **Rule 1:** Final -o → /u/ | casado, alto | casadu, altu | kah-ZAH-doo, AHL-too |
| **Rule 3a:** de → /dji/ | de Miami | dji Miami | jee mee-AH-mee |
| **Rule 4b:** Epenthetic /i/ | York | Yorki | YORK-ee |
| **Rule 5d:** -om → /oun/ | com | coun | KOHN *(nasal)* |
| **Rule 6:** Final L → /u/ | Daniel, Brasil | Danieu, Brasilu | dah-nee-EH-oo, brah-ZEE-loo |
| **Tilde words:** No change | São | São | SOW |

### No changes (feminine forms):
- Sofia, professora, americana, brasileira, casada, alta
- Feminine forms ending in **-a** remain unchanged

## Why This Matters

Understanding substitution mode helps learners:
1. **Hear** the actual pronunciation vs. what's written
2. **Recognize** the systematic patterns (masculine -o always → u → "oo")
3. **Predict** how new words will sound
4. **Practice** pronunciation with confidence

The phonetic realization shows that Portuguese pronunciation follows predictable rules, making it easier to learn once you understand the patterns.

## Format Choice

Both presentation formats are valid:
- **Simple format** is compact and authentic (similar to colloquial texting)
- **Dictionary-style** is professional and familiar to language learners
- **Same rules apply** - only the presentation differs
