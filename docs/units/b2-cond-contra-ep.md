## Outcomes

After this unit, the student can:

- Construct **Type 3 (counterfactual past)** conditional sentences in EP with both register-appropriate result-clause forms
- Apply the **register split** that defines EP Type 3: formal/written EP uses condicional composto (*teria + particípio*); spoken/informal EP commonly substitutes mais-que-perfeito do indicativo (*tinha + particípio*)
- Distinguish *Se eu tivesse estudado, teria passado* (formal) from *Se eu tivesse estudado, tinha passado* (colloquial), recognizing both as productive native EP

## Vocabulary

- **se** — if (counterfactual past reading)
- *Se*-clause forms (review from `b2-mais-que-perfeito-subj`): *tivesse / tivesses / tivesse / tivéssemos / tivessem + particípio*
- Main-clause forms — TWO productive paradigms:
  - **Condicional composto** (formal — see `b2-condicional-composto-ep`): *teria / terias / teria / teríamos / teriam + particípio*
  - **Mais-que-perfeito do indicativo** (colloquial — see `b1-mais-que-perfeito-ind`): *tinha / tinhas / tinha / tínhamos / tinham + particípio*

## Grammar

Type 3 covers situations the speaker treats as **counterfactual** — what *would have* happened *if* something *had been* different in the past. The past time anchor closes the door; the counterfactual is no longer reachable.

The *se*-clause is fixed: **mais-que-perfeito do subjuntivo** (*tivesse + particípio*). The main clause has **two productive forms** in EP, distributed by register.

### Pattern A — Formal register

`se + tivesse + particípio + teria + particípio`

- *Se eu **tivesse estudado** mais, **teria passado** no exame.*
- *Se ela **tivesse vindo** ontem, **teríamos comemorado** juntos.*
- *Se vocês **tivessem chegado** mais cedo, **teriam visto** o início do filme.*

### Pattern B — Colloquial register

`se + tivesse + particípio + tinha + particípio`

- *Se eu **tivesse estudado** mais, **tinha passado** no exame.*
- *Se ela **tivesse vindo** ontem, **tínhamos comemorado** juntos.*
- *Se vocês **tivessem chegado** mais cedo, **tinham visto** o início do filme.*

### Both patterns are correct EP

Same Native Usage Filter logic as Type 2: native EP speakers fluently use both forms and code-switch by register. Neither is more "correct" — they're register-distributed.

| Register | Main-clause form | Where you'll see it |
|---|---|---|
| Formal / written | Condicional composto (*teria estudado, teria passado, teria ido*) | Newspapers, academic essays, formal correspondence |
| Spoken / informal | Mais-que-perfeito do indicativo (*tinha estudado, tinha passado, tinha ido*) | Conversation, WhatsApp, informal email, spontaneous speech |

In a formal essay, *teria* dominates. In conversation, *tinha* is markedly more natural. Production at B2 should match the register; recognition needs to handle both.

### Exemplar pairs

- *Se eu **tivesse sabido**, **teria avisado** todos.* (formal) — If I had known, I would have warned everyone.
- *Se eu **tivesse sabido**, **tinha avisado** todos.* (colloquial) — Same.
- *Se ele **tivesse dito** a verdade, ninguém **teria ficado** chateado.* (formal)
- *Se ele **tivesse dito** a verdade, ninguém **tinha ficado** chateado.* (colloquial)
- *Se a Marta **tivesse vindo**, a festa **teria sido** melhor.* (formal)
- *Se a Marta **tivesse vindo**, a festa **tinha sido** melhor.* (colloquial)

### Mixed conditionals

A common variant: past condition → present consequence (Type 3 *se*-clause + Type 2 main clause):

- Formal: *Se eu **tivesse estudado**, **estaria** a trabalhar como engenheiro hoje.*
- Colloquial: *Se eu **tivesse estudado**, **estava** a trabalhar como engenheiro hoje.*

The *se*-clause stays mais-que-perfeito do subjuntivo (past condition); the main clause shifts to either condicional simples (formal) or imperfeito do indicativo (colloquial) because the result is in the present.

## Traps

- **The colloquial substitute *tinha + particípio* is grammatically the mais-que-perfeito do indicativo.** Same form, different function depending on context. *Tinha estudado* in a Type 3 conditional main clause = condicional composto substitute. *Tinha estudado* in a past-sequence ("when I arrived, he had already studied") = pluperfect. Context determines which reading.
- **Don't translate the BP rule into EP.** A BP-trained student will produce *Se eu tivesse estudado, teria passado* exclusively. That's correct in formal EP but sounds stiff in conversation. The natural EP spoken form is *Se eu tivesse estudado, tinha passado*. Adjust register on entry.
- **The *se*-clause stays subjuntivo.** Don't substitute mais-que-perfeito do indicativo there: *Se eu tinha estudado, tinha passado* is wrong (the *se*-clause needs subjuntivo). The substitution is main-clause only.
- **Type 2 vs Type 3 substitutes use different forms.** Type 2 main-clause substitute = imperfeito do indicativo (*viajava*). Type 3 main-clause substitute = mais-que-perfeito do indicativo (*tinha viajado*). Don't conflate the two register-shift patterns.
- **Mesoclisis in formal main clauses.** Formal EP can produce *Tê-lo-ia feito* — recognize but don't drill. Colloquial EP uses *Tinha-o feito* with normal enclisis.
- **Particípios irregulares.** Same set as the rest of the compound system: *dito, feito, posto, visto, escrito, aberto, ganho, gasto*.
