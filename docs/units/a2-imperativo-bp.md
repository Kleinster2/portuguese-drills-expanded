## Outcomes

- Give direct commands in Portuguese
- Form regular imperative (flip vowel rule)
- Use common irregular imperatives
- Make negative commands (não + imperative)
- Give polite instructions with "por favor"

## Vocabulary

- **Common command verbs:**
  - falar (to speak) → Fale! (Speak!)
  - comer (to eat) → Coma! (Eat!)
  - abrir (to open) → Abra! (Open!)
  - fechar (to close) → Feche! (Close!)
  - esperar (to wait) → Espere! (Wait!)
  - vir (to come) → Venha! (Come!)
  - ir (to go) → Vá! (Go!)
  - fazer (to do) → Faça! (Do!)
  - sentar (to sit) → Sente-se! (Sit down!)
  - levantar (to get up) → Levante-se! (Get up!)

## Grammar

- **Imperative mood - Commands:**
  - Used to give orders, instructions, requests
  - Different forms for você, vocês, and nós
  - This unit focuses on **você** form (most common)

- **Forming você commands:**

  **For -AR verbs:** Change -a to -e
  - falar → você fala → **Fale!** (Speak!)
  - comprar → você compra → **Compre!** (Buy!)
  - fechar → você fecha → **Feche!** (Close!)

  **For -ER/-IR verbs:** Change -e to -a
  - comer → você come → **Coma!** (Eat!)
  - abrir → você abre → **Abra!** (Open!)
  - escrever → você escreve → **Escreva!** (Write!)

- **Pattern: Flip the present tense vowel!**
  - -AR verbs: -a → -e
  - -ER/-IR verbs: -e → -a

- **Negative commands:**
  - Add **NÃO** before the verb:
  - **Não fale!** (Don't speak!)
  - **Não coma!** (Don't eat!)
  - **Não abra!** (Don't open!)

- **Common irregular imperatives:**
  - **ir** → **Vá!** (Go!)
  - **vir** → **Venha!** (Come!)
  - **fazer** → **Faça!** (Do!)
  - **ser** → **Seja!** (Be!)
  - **ter** → **Tenha!** (Have!)
  - **dar** → **Dê!** (Give!)
  - **estar** → **Esteja!** (Be!)

### Exemplar sentences

- Fale português! (Speak Portuguese!)
- Coma devagar! (Eat slowly!)
- Venha aqui! (Come here!)
- Vá para casa! (Go home!)
- Não fale alto! (Don't speak loudly!)
- Abra a porta, por favor. (Open the door, please.)
- Espere um momento! (Wait a moment!)
