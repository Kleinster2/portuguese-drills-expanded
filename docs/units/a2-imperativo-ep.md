## Outcomes

After this unit, the student can:

- Form the **tu-imperative affirmative** (bare 3rd-person singular indicative form: *Fala! Come! Parte!*)
- Form the **tu-imperative negative** using the present subjunctive (*Não fales! Não comas! Não partas!*)
- Form the **você-imperative** (formal, same as BP; uses present subjunctive for both affirmative and negative)
- Recognize the irregular tu-imperative forms (*sê, vai, vem, faz, diz, põe, vê, sai, dá*)

## Vocabulary

Common imperatives in EP:

- **Olha!** — Look! (tu)
- **Vem cá!** — Come here! (tu)
- **Espera!** — Wait! (tu)
- **Diz-me!** — Tell me! (tu)
- **Faça favor!** — Please / Excuse me. (você, formal)
- **Tenha cuidado!** — Be careful! (você, formal)

## Grammar

EP has a **fully productive tu-imperative** (parallel to its tu-pronoun system) plus the você-imperative for formal address.

### Tu-imperative — affirmative

The form equals the **3rd-person singular present indicative** (drop the *-s* from the tu-form):

| Verb | Tu-pres-ind | Imperativo afirmativo (tu) |
|---|---|---|
| falar | (tu) falas | Fala! |
| comer | (tu) comes | Come! |
| partir | (tu) partes | Parte! |
| abrir | (tu) abres | Abre! |

### Irregular tu-imperatives (affirmative)

| Verb | Tu-imperative |
|---|---|
| ser | Sê! |
| ir | Vai! |
| ter | Tem! |
| vir | Vem! |
| fazer | Faz! |
| dizer | Diz! |
| pôr | Põe! |
| ver | Vê! |
| sair | Sai! |
| dar | Dá! |
| estar | Está! / Estás (rare) |

### Tu-imperative — negative

Negative imperative uses the **present subjunctive** of tu (-AR → -es; -ER/-IR → -as):

| Verb | Imperativo negativo (tu) |
|---|---|
| falar | Não fales! |
| comer | Não comas! |
| partir | Não partas! |
| ser | Não sejas! |
| ir | Não vás! |
| ter | Não tenhas! |
| fazer | Não faças! |
| dizer | Não digas! |

### Você-imperative (formal)

Você-imperative uses the **present subjunctive** for both affirmative and negative — the same form for both. No -s ending.

| Verb | Você-imperative (afirm. + neg.) |
|---|---|
| falar | Fale! / Não fale! |
| comer | Coma! / Não coma! |
| partir | Parta! / Não parta! |
| ser | Seja! / Não seja! |
| ir | Vá! / Não vá! |
| fazer | Faça! / Não faça! |

### Exemplar sentences

- *Fala mais alto, por favor!* — Speak louder, please! (tu, affirmative)
- *Não fales tão alto!* — Don't speak so loudly! (tu, negative)
- *Vai ali e espera por mim.* — Go there and wait for me. (tu, affirmative)
- *Não vás sozinho!* — Don't go alone! (tu, negative)
- *Tem cuidado!* — Be careful! (tu, affirmative)
- *Faça favor de esperar.* — Please wait. (você, polite request)
- *Diz-me a verdade.* — Tell me the truth. (tu + clitic)
- *Não me digas isso!* — Don't tell me that! (tu negative + proclitic clitic)

### Clitics with imperatives

- **Affirmative imperative:** clitic enclitic with hyphen: *Diz-me!* (tu), *Diga-me!* (você), *Levanta-te!* (tu reflexive)
- **Negative imperative:** clitic proclitic (negação triggers proclisis): *Não me digas!*, *Não me diga!*, *Não te levantes!*

## Traps

- **Tu-imperative ≠ tu present indicative.** *Tu falas* (you speak) drops the *-s* to become *Fala!* (Speak!). Beginners often say *Falas!* — that's a question intonation or statement, not an imperative.
- **Affirmative and negative tu-imperatives use DIFFERENT forms.** *Fala!* (affirmative — bare 3sg pres ind) vs *Não fales!* (negative — pres subj 2sg). They look similar but differ by the -s and the negation.
- **BP doesn't use the tu-imperative system productively.** BP uses the você-imperative (subjunctive form) almost exclusively even with informal address: *Fale!* (BP all-purpose) vs EP *Fala!* (tu, informal). A BP-trained student must learn the new tu-paradigm for EP.
- **Some irregular forms are easy to confuse.** *Sê* (ser, with circumflex) vs *Se* (conjunction "if") — different meanings. *Põe* (pôr) is distinct from *Pôe* (no such word) and *Pões* (tu pres ind).
- **Mesoclisis exists in imperative + clitic for tu-future commands** (very rare): *Dar-mo-ás!* (You will give it to me!) — recognize, don't produce.
