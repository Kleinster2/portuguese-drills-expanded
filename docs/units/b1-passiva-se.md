## Outcomes

After this unit, the student can:

- Use the **passive-se** construction for generic statements, instructions, commercial signs, and impersonal contexts where no agent is mentioned
- Apply the prescriptive verb-agreement rule (verb agrees in number with the promoted object) and recognize the colloquial-usage exception (singular verb regardless of object number)
- Distinguish passive-se from reflexive-se from reciprocal-se from impersonal-se — same clitic, four functions
- Choose between ser-passive and passive-se based on register and whether the agent is even potentially mentionable

## Vocabulary

- **se** — clitic marking passive/impersonal use (same form as reflexive *se*, distinct function)
- Common passive-se verbs: *vender, alugar, comprar, procurar, falar, fazer, escrever, dizer, viver*

## Grammar

The passive-se construction promotes a direct object to the position of grammatical subject and uses the clitic *se* in place of an agent-marked passive auxiliary. The agent is **never expressible** in this construction — that's its core distinguishing feature.

### Pattern

`verbo-conjugado + se + objeto-direto-promovido`

- *Vendem-se livros.* — Books are sold (here / by us, no agent mentionable).
- *Aluga-se apartamento.* — Apartment for rent.
- *Procura-se programador.* — Programmer wanted.

### Verb agreement — prescriptive vs colloquial

Prescriptive grammar: verb agrees in number with the promoted object.

| Object number | Prescriptive verb form |
|---|---|
| Singular | *Vende-se casa.* |
| Plural | *Vendem-se casas.* |

Colloquial usage (heard in BP and EP both): verb stays singular regardless of object number.

| Colloquial form | Status |
|---|---|
| *Vende-se casas.* | Heard frequently; considered substandard by traditional grammars |
| *Aluga-se apartamentos.* | Same — substandard but common |

The prescriptive form is what to teach for production; the colloquial form is what students will encounter on real shop signs and news headlines.

### Common contexts

| Context | Example |
|---|---|
| Commercial signs | *Vende-se / Aluga-se / Trespassa-se* |
| Job ads | *Procura-se cozinheiro com experiência.* |
| Generic statements | *Aqui se fala português.* / *Não se fuma na escola.* |
| News headlines | *Discutiu-se a proposta no parlamento.* |
| Instructions | *Misturam-se os ingredientes lentamente.* |
| Impersonal generalizations | *Vive-se bem em Lisboa.* (impersonal-se, often grouped with passive-se) |

### Passive-se vs ser-passive — when to choose which

| Use ser-passive when… | Use passive-se when… |
|---|---|
| Agent matters or might be mentioned | Agent is irrelevant or unknown |
| Formal / written register | Generic / commercial / instruction context |
| Specific affected entity in focus | Generic / class-of-things in focus |

Examples:

- *O livro foi escrito **por ela**.* (ser-passive — agent named) → can't switch to passive-se without losing the agent.
- *Vendem-se livros usados.* (passive-se — generic, no agent) → switching to ser-passive (*Livros usados são vendidos*) sounds odd; the passive-se carries the commercial-sign register.

### Exemplar sentences

- *Vendem-se livros usados.* — Used books for sale.
- *Aluga-se quarto a estudantes.* — Room for rent for students.
- *Procura-se cozinheiro experiente.* — Experienced cook wanted.
- *Aqui se fala português, espanhol e inglês.* — Portuguese, Spanish, and English spoken here.
- *Não se permite fumar nesta sala.* — Smoking is not permitted in this room.
- *Discutiram-se vários assuntos na reunião.* — Several matters were discussed at the meeting.
- *Vive-se bem nesta cidade.* — One lives well in this city. / Life is good in this city. (impersonal-se reading)

## Traps

- **Don't confuse passive-se with reflexive-se.** Same clitic form, different function. Reflexive: *Ela se chama Maria* / *Chama-se Maria* (the subject acts on themselves). Passive: *Vendem-se livros* (no subject acts; the verb describes what happens to the object). Context disambiguates.
- **Don't try to add an agent.** *Vende-se a casa pelo proprietário* is wrong — the construction structurally excludes agents. If you want to mention the agent, use ser-passive: *A casa é vendida pelo proprietário*.
- **Verb agreement is variable in real usage but prescriptive in classroom Portuguese.** Drill the prescriptive form (*vendem-se livros*) for production; recognize the colloquial form (*vende-se livros*) for comprehension. On exam-grade writing, always agree.
- **Clitic placement applies to the *se* per variant rules.** BP often uses proclisis with passive-se (*Aqui se fala português*); EP defaults to enclisis (*Fala-se português aqui*). Both forms exist in both variants depending on triggers — see [[a2-reflexivos-bp]] / [[a2-reflexivos-ep]] for the full clitic-placement rules. The *se* in passive-se follows those same rules.
- **Some verbs resist passive-se.** Intransitive verbs can't take it. Verbs whose direct object is a person typically prefer ser-passive (*Foi visto pelo polícia* > *Viu-se ele*). The construction is most natural with concrete-noun direct objects (*livros, casas, carros, apartamentos*).
- **Impersonal-se reading.** When the verb has no direct object, *se* makes the construction impersonal: *Vive-se bem aqui* (one lives well here / life is good here). Some grammars classify this separately; pedagogically it sits naturally with passive-se as part of the same agentless-construction family.
