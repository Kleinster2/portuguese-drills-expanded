## Outcomes

After this unit, the student can:

- Use direct object pronouns (DOP) in EP with correct clitic placement
- Apply the same enclisis-as-default + proclisis-trigger rules covered in `a2-reflexivos-ep` to the full DOP set
- Handle the *-r + lo/la* contraction (*comer + o → comê-lo*) and the *-m / -ão + lo/la* allomorphy (*comem + o → comem-no*)

## Vocabulary

Direct object pronouns:

| Pessoa | DOP |
|---|---|
| eu | me |
| tu | te |
| ele / o (m) | o |
| ela / a (f) | a |
| nós | nos |
| eles / os (m) | os |
| elas / as (f) | as |

(*Vos* exists but is essentially dead in modern EP; use *vocês* with the standard 3pl forms *os/as*.)

## Grammar

DOP placement follows the same rules as reflexive clitics (see `a2-reflexivos-ep`): **enclisis is the default**, **proclisis is triggered** by negação, conjunções subordinativas, advérbios de inclusão, pronomes interrogativos/relativos/indefinidos, and **mesoclisis** appears in futuro / condicional.

### Default enclisis

- *Vejo-o todos os dias.* — I see him every day.
- *Comprámo-las ontem.* — We bought them yesterday.
- *Conheces-me?* — Do you know me?

### Proclisis (triggered)

- *Não o vejo há semanas.* — I haven't seen him in weeks. (negação)
- *Disse que a comprou.* — He said he bought it. (conjunção *que*)
- *Quem te ajudou?* — Who helped you? (interrogativo)
- *Também o conheço.* — I know him too. (advérbio de inclusão)

### Mesoclisis (futuro / condicional, formal)

- *Vê-lo-ei amanhã.* — I will see him tomorrow.
- *Comprá-la-ia se pudesse.* — I would buy it if I could.

In modern spoken EP, replaced by *vou + infinitivo + clitic-on-infinitive*: *Vou vê-lo amanhã*.

### -r + lo/la contraction

When the verb ends in **-r, -s, or -z**, those consonants drop and the clitic *o/a/os/as* takes the form **lo/la/los/las**:

| Verb | + DOP | Result |
|---|---|---|
| comer + o | comê-lo | (eat it) |
| fazer + as | fazê-las | (do them) |
| comprar + os | comprá-los | (buy them) |
| diz + o | di-lo | (says it) |
| fizemos + a | fizemo-la | (we did it) |

The vowel often gets a circumflex (-ê-, -â-) to mark the stress shift.

### -m / -ão + o/a → -m + no/na

When the verb ends in a nasal (-m, -ão, -ões, -õe), the clitic takes the form **no/na/nos/nas**:

| Verb | + DOP | Result |
|---|---|---|
| comem + o | comem-no | (they eat it) |
| põe + a | põe-na | (puts it) |
| dão + os | dão-nos | (they give them) |

### Exemplar sentences

- *Vejo o João todos os dias. Vejo-o todos os dias.* — I see him every day.
- *Não comprei o livro. Não o comprei.* — I didn't buy the book.
- *A Maria conhece-te?* — Does Maria know you?
- *Quero comer o bolo. Quero comê-lo.* — I want to eat it.
- *Os meus amigos compraram-nos.* — My friends bought them.
- *Quando o vires, diz-lhe que ligo.* — When you see him, tell him I'll call.

## Traps

- **BP frequently uses subject pronouns where EP uses DOP clitics.** BP: *Eu vi ele ontem* (subject pronoun in object position — colloquial). EP: *Vi-o ontem.* (correct DOP). EP doesn't tolerate subject pronouns as objects in formal/standard register.
- **The -r + lo/la contraction is invisible to learners at first.** Many beginners write *comer-o* instead of *comê-lo*. The dropped *-r* and the changed clitic form (*lo* not *o*) are linked — both rules fire together.
- **Mesoclisis is a recognition target, not a production target for A2.** Don't drill *vê-lo-ei* — it's archaic in spoken use. Recognize it in formal text; produce *vou vê-lo* in conversation.
- **EP and BP both have *o/a/os/as* but place them differently.** Same form, different rules. The form difference is in tu (EP *te* vs BP also *te* but rarely used as default informal address).
- **Don't double-mark with subject + clitic.** *O João, ele viu-me* — the doubling sounds heavy in EP. Use *O João viu-me* or just *Ele viu-me*.
