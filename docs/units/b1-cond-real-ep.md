## Outcomes

After this unit, the student can:

- Construct **Type 1 (real / likely)** conditional sentences in EP using `se + presente indicativo + presente / futuro indicativo`
- Distinguish habitual conditionals (general truths) from specific-future real conditionals
- Recognize that Type 1 does NOT use subjuntivo — that's Type 2 (irreal) and Type 3 (counterfactual past)

## Vocabulary

- **se** — if
- Common temporal anchors: *agora, sempre, normalmente, todos os dias, amanhã, na próxima semana*

## Grammar

Type 1 covers conditions the speaker treats as **real, likely, or habitual** — the situation may or may not happen, but isn't being imagined as contrary to fact.

Two main EP patterns (essentially the same as BP):

### Pattern 1a — Habitual / general

`se + presente indicativo + presente indicativo`

- *Se chove, fico em casa.* — If it rains, I stay home. (general / habitual)
- *Se ele telefona, eu atendo.* — If he calls, I answer.
- *Se estudas, passas no exame.* — If you study, you pass the exam. (timeless truth, tu-form)

### Pattern 1b — Specific future (real)

`se + futuro do subjuntivo + presente / futuro indicativo`

In EP (as in BP), the *se*-clause for specific future events takes **futuro do subjuntivo**. For regular verbs the form is identical to the infinitive, which is why this often *looks* like indicative:

- *Se chover amanhã, fico em casa.* — If it rains tomorrow, I'll stay home.
- *Se ele telefonar, eu atendo.* — If he calls, I'll answer.
- *Se tu estudares, passas no exame.* — If you study, you'll pass the exam. (tu-form)

### Exemplar set

- *Se eu tenho tempo, cozinho.* — If I have time, I cook. (habitual)
- *Se eu tiver tempo amanhã, cozinho.* — If I have time tomorrow, I'll cook. (specific future)
- *Se chover, fico em casa.* — If it rains, I'll stay home.
- *Se a Marta vier, ficamos mais um bocado.* — If Marta comes, we'll stay a bit longer.
- *Se quiseres mais informação, telefona-me.* — If you want more information, call me. (tu-form, with clitic enclisis on imperative)

### Type 1 is essentially shared with BP

The grammar is the same; what differs is the surrounding usage:

- EP uses *tu* and tu-conjugations: *Se tu estudares, passas* (vs BP *Se você estudar, passa*).
- EP imperative carries the tu-form: *telefona-me* (vs BP *me telefone*).
- The Pattern 1b futuro do subjuntivo is identical in form for regulars; irregulars (*tiver, fizer, vier, vir*) are also shared.

## Traps

- **Don't use subjuntivo presente in Type 1.** *Se ele venha, eu ajudo* — wrong (subjuntivo presente). Type 1 takes indicativo or futuro do subjuntivo. The hypothetical/wish reading would be Type 2: *Se ele viesse, eu ajudaria*.
- **Pattern 1a vs 1b is a tense-of-the-se-clause distinction.** Habitual takes presente indicativo; specific-future takes futuro do subjuntivo. The forms can look identical for regular verbs (*chover → chove* vs *chover → chover*) but differ for irregulars (*ter → tem* vs *ter → tiver*).
- **EP-specific: tu-form in the *se*-clause and main clause.** *Se tu vens, eu fico* (presente indicativo, tu-form). *Se tu vieres, eu fico* (futuro do subjuntivo, tu-form). BP would default to *se você vem* / *se você vier*.
- **Type 1 is NOT the place for *se eu fosse rico*.** That's Type 2 (irreal). Type 1 is *se eu sou / se eu for*, indicativo-only.
- **Imperative in main clause.** *Se quiseres, vem cá* — tu-imperative *vem* triggered as natural EP advice/invitation form. Type 1 + imperative is a common pairing in EP.
