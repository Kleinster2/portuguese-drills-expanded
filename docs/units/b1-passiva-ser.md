## Outcomes

After this unit, the student can:

- Transform active sentences into the analytic passive (**ser + particípio**) preserving tense, person, and agreement
- Introduce the agent with **por / pela / pelo / pelos / pelas** when needed, or omit it when irrelevant
- Apply past-participle agreement with the grammatical subject (gender + number)
- Choose between active and passive based on whether the speaker wants to foreground the agent or the affected entity

## Vocabulary

- **ser** — auxiliary for the analytic passive (conjugates in any tense)
- **por / pela / pelo / pelos / pelas** — agent introducer (*por* + article contractions)
- Past participles — review the irregular set: *dito, feito, posto, visto, escrito, aberto, ganho, gasto, aceito/aceitado, eleito/elegido*

## Grammar

The analytic passive transforms an active sentence by:
1. Promoting the direct object to grammatical subject
2. Conjugating *ser* in the corresponding tense
3. Adding the past participle of the main verb (agreeing in gender + number with the new subject)
4. Optionally introducing the original subject as agent with *por*

### Active → passive transformation

| Active | Passive |
|---|---|
| A Ana fez o bolo. | O bolo foi feito (pela Ana). |
| Os engenheiros construíram a ponte. | A ponte foi construída (pelos engenheiros). |
| O autor escreveu estas cartas. | Estas cartas foram escritas (pelo autor). |

### Tense correspondence

The auxiliary *ser* takes whichever tense the active verb had:

| Active tense | Passive form |
|---|---|
| Presente: *escreve* | é escrito/a |
| Pretérito: *escreveu* | foi escrito/a |
| Imperfeito: *escrevia* | era escrito/a |
| Futuro: *escreverá* | será escrito/a |
| Condicional: *escreveria* | seria escrito/a |
| Pretérito composto: *tem escrito* | tem sido escrito/a |
| Subjuntivo presente: *(que) escreva* | (que) seja escrito/a |
| Subjuntivo imperfeito: *(se) escrevesse* | (se) fosse escrito/a |

### Participle agreement

The participle agrees with the **grammatical subject** of the passive sentence (the promoted object):

- *O livro foi **escrito** pela Ana.* — masc sg
- *A carta foi **escrita** pela Ana.* — fem sg
- *Os livros foram **escritos** pela Ana.* — masc pl
- *As cartas foram **escritas** pela Ana.* — fem pl

### Agent introduction

Use *por* + article contractions for the agent:

- *por + o = pelo*
- *por + a = pela*
- *por + os = pelos*
- *por + as = pelas*

Examples:
- *O bolo foi comprado **pelo** padeiro.*
- *A casa foi construída **pela** empresa.*
- *Os documentos foram assinados **pelos** advogados.*
- *As fotos foram tiradas **pela** Maria.*

The agent is **optional**. Omit it when irrelevant or unknown:

- *O livro foi publicado em 2020.* — agent unmentioned
- *As janelas foram partidas durante a tempestade.* — agent unmentioned

### Exemplar sentences

- *Esta canção **foi composta** em 1978.* — This song was composed in 1978.
- *Os documentos **serão entregues** amanhã.* — The documents will be delivered tomorrow.
- *O presidente **foi eleito** em outubro.* — The president was elected in October.
- *As decisões **são tomadas** pela direção.* — The decisions are made by management.
- *A obra **tinha sido feita** antes do incêndio.* — The work had been done before the fire.
- *É importante que a verdade **seja dita**.* — It's important that the truth be told. (subjuntivo presente)

## Traps

- **Forgetting participle agreement is the most common error.** *O livro foi feita* (wrong — *feito* should be masc sg). The participle's gender/number tracks the grammatical subject, not the agent.
- **Por vs pelo/pela/pelos/pelas.** *Por João* is wrong if João takes an article (*O João foi visto*). *Pelo João foi feito o trabalho* — pelo because *o João* contracts. Apply normal *por +* article contractions.
- **Don't overuse passive in casual Portuguese.** Spoken Portuguese (BP and EP both) prefers active constructions much more than English does. *O livro foi escrito por ele* sounds heavy compared to *Ele escreveu o livro*. Reserve passive for when the agent really doesn't matter or when the affected entity is the topic.
- **Don't conflate ser-passive with estar-passive (estar + particípio).** *O livro foi escrito* (action: someone wrote it) vs *O livro está escrito* (state: it's currently in written form). Different aspects entirely. The unit covers ser-passive only; estar + particípio is a separate construction.
- **Double participles matter here.** Some verbs have both regular and irregular participles, and the passive uses the irregular one: *aceitar → aceito* (passive: *foi aceito*, NOT *foi aceitado*). Same with *eleger → eleito* (passive: *foi eleito*, NOT *foi elegido*). The unit `b2-participios-duplos` (Phase 2 placeholder, future enrichment) catalogs the full pair list.
- **Particípios irregulares anchor here.** This unit assumes participle forms from prior compound-tense exposure (`b1-mais-que-perfeito-ind`, `b1-pret-perfeito-composto-bp/ep`). A dedicated participle-introduction unit doesn't yet exist in the corpus — backlog item.
