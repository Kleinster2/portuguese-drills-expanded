## Outcomes

After this unit, the student can:

- Use conditional conjunctions beyond *se* — **caso, a menos que, a não ser que, contanto que, desde que** — with appropriate subjuntivo
- Distinguish the **conditional reading** of *desde que* (subjuntivo, "provided that") from its **temporal reading** (indicative, "since [time]")
- Choose between *se* and the more nuanced conjunctions based on the speaker's stance toward the condition (real vs hypothetical, restrictive, exclusionary, etc.)

## Vocabulary

The conditional-conjunction inventory beyond *se*:

| Conjunction | Meaning | Mood | Stance |
|---|---|---|---|
| **caso** | in case / if | subjuntivo | hypothetical, slightly more formal than *se* |
| **a menos que** | unless | subjuntivo | exclusionary |
| **a não ser que** | unless | subjuntivo | exclusionary, slightly more formal |
| **contanto que** | provided that | subjuntivo | restrictive, as a guarantee |
| **desde que** (conditional) | provided that | subjuntivo | restrictive |
| **desde que** (temporal) | since [time] | indicativo | temporal sequence |
| **salvo se** | except if | indicativo or subjuntivo | exclusionary, formal |
| **se acaso** | if by chance | subjuntivo | tentative |
| **na hipótese de** + infinitivo | in case of | infinitivo | formal |

## Grammar

These conjunctions extend the conditional system beyond plain *se*. Each carries its own pragmatic stance:

- **caso** — neutral conditional, slightly more formal than *se*
- **a menos que / a não ser que** — exclusionary ("unless")
- **contanto que / desde que** — restrictive ("provided that")
- **desde que** also has a temporal reading (indicative)

This unit builds on the conditional-sentence framework taught in [[b1-cond-real-bp]] / [[b1-cond-real-ep]] (Type 1), [[b2-cond-irreal-bp]] / [[b2-cond-irreal-ep]] (Type 2), and [[b2-cond-contra-bp]] / [[b2-cond-contra-ep]] (Type 3). The form-anchor for condicional simples is [[b1-condicional]].

### Caso — alternative to se

*Caso* is essentially "in case / if" — slightly more formal than *se* but pragmatically similar:

- *Caso **chova**, ficamos em casa.* — In case it rains, we'll stay home.
- *Caso **precises** de ajuda, liga-me.* — If you need help, call me.
- *Caso **tivesses estudado**, terias passado.* — If you had studied, you would have passed. (Type-3 with caso instead of se)

*Caso* always takes subjuntivo. It cannot replace *se* in Type-1 indicative-only contexts (no *caso chove* with present indicative).

### A menos que / a não ser que — unless

These exclusionary conjunctions take subjuntivo:

- *Vou à festa, **a menos que chova**.* — I'll go to the party, unless it rains.
- *Não saio, **a menos que tu venhas comigo**.* — I'm not going out unless you come with me.
- *Aceitarei o emprego, **a não ser que apareça** algo melhor.* — I'll accept the job, unless something better comes up.

The exclusionary stance: the main-clause action holds **except** in the case described by the subordinate. *A menos que* and *a não ser que* are interchangeable; the latter is slightly more formal.

### Contanto que / desde que (conditional) — provided that

These restrictive conjunctions establish a condition as a guarantee:

- *Vou contigo, **contanto que cheguemos** cedo.* — I'll go with you, provided that we arrive early.
- *Podes sair, **contanto que voltes** antes da meia-noite.* — You can go out, provided you come back before midnight.
- *Podes ficar, **desde que estudes**.* — You can stay, provided that you study.

The restrictive stance: the main-clause action holds **only on condition that** the subordinate clause is met.

### Desde que — conditional vs temporal

This is the highest-leverage trap in the unit. *Desde que* has two readings, distinguished by mood:

| Reading | Mood | Example |
|---|---|---|
| Conditional ("provided that") | subjuntivo | *Podes ficar **desde que estudes**.* — You can stay provided you study. |
| Temporal ("since [time]") | indicativo | *Conheço-o **desde que nasceu**.* — I've known him since he was born. |

The temporal reading anchors a starting point in time; the conditional reading establishes a restriction. Same surface conjunction, different moods, different meanings.

### Salvo se / a não ser que — except if

*Salvo se* mostly takes indicative when describing real exceptions, subjuntivo for hypothetical exceptions:

- *Vou **salvo se chove muito**.* — I'll go except if it's raining a lot. (real exception, indicative)
- *Vou **salvo se chovesse muito**.* — Same with hypothetical past framing, subjuntivo.

In practice, *a não ser que* + subjuntivo is more common and more flexible.

### Tense of subjuntivo follows the main clause

Standard tense agreement:

- Main verb in present/future → presente subjuntivo: *Vou contigo, contanto que **cheguemos** cedo.*
- Main verb in past → imperfeito subjuntivo: *Fui contigo, contanto que **chegássemos** cedo.*
- Counterfactual past → mais-que-perfeito subjuntivo: *Teria ido, contanto que **tivéssemos chegado** cedo.*

### Exemplar set

- *Caso ele **venha**, avisa-me.* — If he comes, let me know.
- *Saio à noite, **a menos que esteja** muito cansado.* — I go out at night, unless I'm very tired.
- *Vou ajudar-te, **contanto que sejas** honesto comigo.* — I'll help you, provided you're honest with me.
- *Podes usar o meu carro, **desde que devolvas** com gasolina.* — You can use my car, provided you return it with fuel.
- *Conheço a Maria **desde que era** criança.* — I've known Maria since she was a child. (temporal — indicative)
- *Conheço a Maria **desde que tu a apresentaste**.* — I've known Maria since you introduced her. (temporal — indicative)
- *Aceito o convite, **a não ser que apareça** algo urgente.* — I accept the invitation, unless something urgent comes up.

## Traps

- **Desde que is the highest-leverage ambiguity.** *Desde que cheguei* is temporal (indicative — "since I arrived"); *desde que chegues* is conditional (subjuntivo — "provided that you arrive"). The mood is the disambiguator.
- **Don't use indicative with caso.** *Caso chove* is wrong; *caso chova* is right. Caso always takes subjuntivo.
- **A menos que and a não ser que require subjuntivo.** *A menos que vens* is wrong; *a menos que venhas* is right.
- **Contanto que vs com tanto que.** Note the spelling: *contanto que* (single word for the conjunction) vs *com tanto que* (preposition + intensifier — different meaning entirely). Same pronunciation, different writing.
- **Salvo se can take either mood.** Indicative for real exceptions, subjuntivo for hypothetical. When in doubt, *a não ser que* + subjuntivo is more flexible and unambiguous.
- **Don't import *se* habits to other conjunctions.** *Caso chove* (wrong — caso requires subj) is a common error from importing the *se* + presente indicativo (Type 1 habitual) pattern into *caso* contexts. The two conjunctions don't share mood behavior.
- **Tense alignment with main clause matters.** Past main clause forces past subjuntivo: *Aceitei o emprego contanto que **pagassem** bem* (past hypothetical) — not *contanto que paguem*.
