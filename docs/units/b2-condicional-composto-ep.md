## Outcomes

After this unit, the student can:

- Form the condicional composto in EP (*teria/terias/teria/teríamos/teriam + particípio*)
- Use it as the main-clause verb in formal/written Type 3 counterfactual past sentences
- Recognize and produce the **colloquial EP substitute**: *tinha + particípio* (mais-que-perfeito do indicativo) instead of *teria + particípio*
- Choose between forms by register — formal/written takes *teria*, spoken/informal often takes *tinha*

## Vocabulary

- **teria / terias / teria / teríamos / teriam + particípio** — formal compound conditional
- **tinha / tinhas / tinha / tínhamos / tinham + particípio** — spoken substitute (form is mais-que-perfeito do indicativo, see `b1-mais-que-perfeito-ind`)

## Grammar

Form: condicional simples of *ter* + particípio passado.

| Pessoa | Auxiliar (formal) | Auxiliar (colloquial) | + particípio |
|---|---|---|---|
| eu | teria | tinha | falado |
| tu | terias | tinhas | falado |
| você / ele / ela | teria | tinha | falado |
| nós | teríamos | tínhamos | falado |
| vocês / eles / elas | teriam | tinham | falado |

The two columns are **interchangeable in colloquial EP**. Both are productive, both are correct; the choice is a register marker.

### Three primary uses (same as BP)

**1. Main clause of Type 3 conditional — see `b2-cond-contra-ep`**

- Formal: *Se eu tivesse estudado mais, **teria passado**.*
- Colloquial: *Se eu tivesse estudado mais, **tinha passado**.*

**2. Past hypothetical consequence (no explicit *se*-clause)**

- Formal: *Eu **teria ido** à festa, mas estava doente.*
- Colloquial: *Eu **tinha ido** à festa, mas estava doente.*

**3. Reported speech / future-in-the-past**

- Formal: *Ele disse que **teria terminado** até hoje.*
- Colloquial: *Ele disse que **tinha terminado** até hoje.*

### Register split: when to use which

| Register | Form | Where you'll see it |
|---|---|---|
| Formal / written | *teria + particípio* | Newspapers, academic writing, formal correspondence |
| Spoken / informal | *tinha + particípio* | Conversation, WhatsApp, informal email, journalism aimed at general readers |

Native EP speakers code-switch fluidly. In a formal essay, *teria* dominates. In conversation, *tinha* is the more natural choice. Production at B2 should match the register; recognition needs to handle both.

### Exemplar sentences

- *Eu **teria comprado** o livro, mas o preço era alto.* — I would have bought the book, but the price was high. (formal)
- *Eu **tinha comprado** o livro, mas o preço era alto.* — Same meaning, conversational.
- *Tu **terias gostado** do filme.* — You would have liked the movie. (formal)
- *Tu **tinhas gostado** do filme.* — Same, spoken.
- *Nós **teríamos ido** se tivéssemos sido convidados.* — We would have gone if we'd been invited. (formal)
- *Nós **tínhamos ido** se tivéssemos sido convidados.* — Same, spoken.

## Traps

- **The substitute *tinha + particípio* is grammatically the mais-que-perfeito do indicativo** (see `b1-mais-que-perfeito-ind`). Same form, different function depending on context. *Tinha estudado* in a Type 3 conditional clause = condicional composto substitute. *Tinha estudado* in a past-sequence ("when I arrived, he had already studied") = pluperfect. Context determines which reading.
- **BP doesn't make this substitution.** A BP speaker hearing *Tinha ido se pudesse* might parse it as past pluperfect (he had gone if he could — strange) instead of as a conditional substitute. EP students should know this is a real BP/EP divergence point.
- **Don't mix registers within one sentence.** *Se eu tivesse podido, teria ido* (formal+formal) or *Se eu tivesse podido, tinha ido* (formal+colloquial substitute) are both natural. *Se eu tivesse podido, ia* mixes too far down the colloquial scale.
- **Particípios irregulares.** Same set as BP — *dito, feito, posto, visto, escrito, aberto, ganho, gasto*.
- **Mesoclisis with clitics in formal EP.** *Tê-lo-ia feito* (I would have done it) — recognize but don't drill at B2. The colloquial form *Tinha-o feito* is far more common.
