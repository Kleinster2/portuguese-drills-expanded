## Outcomes

After this unit, the student can:

- Recognize the conventions of academic Portuguese: formal vocabulary, third-person convention, passive constructions, nominalization
- Apply **hedging** appropriate to academic claims (impersonal subjuntivo + condicional)
- Use citation phrases (*segundo X, conforme aponta Y, na perspetiva de Z*)
- Avoid informal contractions and personal-conversational register in academic writing
- Recognize BP/EP citation-style differences (Brazilian academic convention vs Portuguese academic convention)

## Vocabulary

### Formal vs informal vocabulary register

Academic Portuguese systematically prefers more Latinate, longer-syllable, lower-frequency vocabulary. A short list of common substitutions:

| Informal | Academic |
|---|---|
| usar | utilizar / empregar |
| ajudar | auxiliar / contribuir para |
| pôr | colocar / posicionar |
| dizer | afirmar / declarar / sustentar |
| achar | considerar / entender |
| começar | iniciar / dar início a |
| acabar | finalizar / concluir |
| mostrar | demonstrar / evidenciar |
| ver | observar / constatar / verificar |
| dar | conceder / atribuir / fornecer |
| pegar | obter / adquirir |
| muito | considerável / significativo / substancial |

Academic vocabulary isn't *better* — it's register-marked. A Master's thesis defending *cheguei à conclusão* sounds correct; the same phrase in a casual blog post would feel overweight. Match register to context.

### Citation phrases — introducing sources

- **segundo X** — according to X
- **conforme aponta X** — as X points out
- **conforme demonstra X** — as X demonstrates
- **na perspetiva de X / no entender de X** — in X's view / understanding
- **como argumenta X** — as X argues
- **X (ano) afirma que…** — X (year) states that… (with parenthetical year)
- **na obra clássica de X** — in X's classic work
- **citação direta**: aspas + (X, ano: pp.) — for direct quotations

### Hedging vocabulary — managing claim strength

- **parece que / aparenta-se que** — it seems that
- **é possível que / é provável que** — it's possible / probable
- **tendencialmente** — tendentially
- **em larga medida** — to a large extent
- **em geral / regra geral** — in general / as a general rule
- **com algumas exceções** — with some exceptions
- **talvez** — perhaps
- **eventualmente** (academic Portuguese: "possibly", not English "eventually") — possibly

## Grammar

Academic Portuguese has identifiable grammatical patterns beyond vocabulary register:

### 1. Third-person convention (no "eu" subject)

Academic prose typically avoids first-person *eu* in favor of:

- Impersonal constructions: *Conclui-se que / Verifica-se que / Constata-se que*
- First-person plural (the editorial "we"): *Argumentamos que / Verificamos que / Pretendemos demonstrar que*
- Nominalization: *Esta análise sugere que…* (instead of *Eu analiso e digo que…*)
- Passive: *Foi observado que…* (instead of *Eu observei que…*)

The *eu*-form sounds undergraduate or memoir-like in formal academic register.

### 2. Passive constructions for objectivity

Active voice: *Os investigadores observaram este fenómeno em 2020.*

Passive voice (academic preference): *Este fenómeno foi observado em 2020.* / *Este fenómeno foi observado pelos investigadores em 2020.*

The passive de-emphasizes the agent and emphasizes the affected entity (the phenomenon). In academic discourse, the entity is often the relevant focus; the agent (researchers) is secondary information.

See [[b1-passiva-ser]] for the full passive system.

### 3. Nominalization — verbs as nouns

Academic prose densifies meaning by converting verbs into nouns:

| Verbal version | Nominalized version |
|---|---|
| *Quando os preços aumentam, o consumo diminui.* | *O aumento dos preços provoca a diminuição do consumo.* |
| *Importa que se discuta o tema.* | *A discussão deste tema é importante.* |
| *Quando se observa o fenómeno cuidadosamente…* | *A observação cuidadosa do fenómeno…* |

Nominalization compresses information into noun phrases that can serve as subjects or objects. It sounds heavier but allows more precise relationships in fewer words.

### 4. Hedging through impersonal subjuntivo

Academic claims are typically presented as worth-considering rather than asserted-as-fact. The impersonal subjuntivo (covered in [[b1-subj-trigger-impessoal]]) is the workhorse:

- *É possível **que esta interpretação esteja** equivocada.* — Hedging the assertion
- *Convém notar **que existam** alternativas viáveis.* — Acknowledging without committing
- *Importa salientar **que o estudo apresente** limitações metodológicas.* — Self-criticism appropriate to academic register

### 5. Condicional for tentative formulations

Condicional (see [[b1-condicional]]) softens claims further:

- *Diria que esta abordagem é problemática.*
- *Seria pertinente investigar mais profundamente.*
- *Poderíamos considerar uma perspetiva alternativa.*
- *A teoria proposta beneficiaria de mais validação empírica.*

### Citation conventions — BP vs EP

Citation styles differ slightly between Brazilian and Portuguese academic conventions:

| Aspect | BP convention | EP convention |
|---|---|---|
| Author-date format | (Silva, 2020) | (Silva, 2020) — similar |
| Direct quote attribution | "..." (Silva, 2020, p. 15) | "..." (Silva, 2020: 15) |
| Footnote vs in-text | More in-text in BP | More footnote use in EP |
| Bibliographic style | ABNT (Brazilian) | various (often APA, ISO) |

For B2 production, follow whichever convention the institution requires. Recognize both when reading.

### Exemplar academic paragraph

> *A análise dos dados sugere que a relação entre crescimento económico e emissões de carbono é mais complexa do que sustenta a literatura clássica. **Verifica-se que** países desenvolvidos apresentam, em larga medida, uma estabilização das emissões mesmo com crescimento contínuo. **Conforme aponta** Silva (2022), este fenómeno deve-se à transição estrutural dos setores económicos, mais do que a políticas ambientais explícitas. **É possível que** a relação se altere em economias emergentes, **embora** os dados disponíveis ainda **sejam** inconclusivos. **Diria que** uma investigação longitudinal seria pertinente.*

The paragraph combines: nominalization (*A análise dos dados*), impersonal verbs (*verifica-se*), citation (*conforme aponta*), hedging subjuntivo (*é possível que se altere*, *embora sejam*), condicional (*diria que, seria pertinente*).

## Traps

- **Informal contractions sneaking in.** *tá / tô / pra* in academic writing immediately mark the register as inappropriate. Always *está / estou / para* in formal text.
- **Over-hedging or under-hedging.** Too many *é possível que / talvez / parece que* in succession sounds evasive — the reader can't tell what the writer believes. Too few sounds dogmatic — the writer overcommits to claims that should be qualified. Balance.
- **First-person *eu* in academic writing.** Generally avoid; use first-person plural (*argumentamos que*) or impersonal (*conclui-se que*). Some disciplines (especially in BP) increasingly accept *eu* in qualitative research; check journal conventions.
- **Eventualmente means "possibly", NOT "eventually".** *Eventualmente teremos uma resposta* in EP/BP academic prose means "we might possibly have an answer", not "in the end we'll have an answer." False friend with English.
- **Citation-style mixing.** A single text shouldn't mix Brazilian (Silva, 2020, p. 15) and Portuguese (Silva, 2020: 15) conventions. Pick one and apply consistently.
- **Nominalization overdose.** *A nominalização excessiva da expressão verbal causa a densificação da prosa académica para além do limite da legibilidade.* — Technically correct, but the cascade of noun phrases is exhausting. Use nominalization to compress, not to obscure.
