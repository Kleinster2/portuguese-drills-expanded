## Outcomes

After this unit, the student can:

- Construct **Type 1 (real / likely)** conditional sentences in BP using the indicative pair: `se + presente indicativo + presente / futuro indicativo`
- Distinguish habitual conditionals (general truths) from specific-future real conditionals
- Recognize that Type 1 does NOT use subjuntivo — that's Type 2 (irreal) and Type 3 (counterfactual past)

## Vocabulary

- **se** — if
- Common temporal anchors: *agora, sempre, normalmente, todo dia, amanhã, na próxima semana*

## Grammar

Type 1 covers conditions the speaker treats as **real, likely, or habitual** — the situation may or may not happen, but isn't being imagined as contrary to fact.

Two main BP patterns:

### Pattern 1a — Habitual / general

`se + presente indicativo + presente indicativo`

- *Se chove, eu fico em casa.* — If it rains, I stay home. (general / habitual)
- *Se ele liga, eu atendo.* — If he calls, I answer.
- *Se você estuda, você passa.* — If you study, you pass. (timeless truth)

### Pattern 1b — Specific future (real)

`se + futuro do subjuntivo + presente / futuro indicativo`

In BP, the *se*-clause for specific future events takes **futuro do subjuntivo** (covered in `b2-subj-futuro` — for regular verbs the form is identical to the infinitive, which is why this often *looks* like indicative):

- *Se chover amanhã, vou ficar em casa.* — If it rains tomorrow, I'll stay home.
- *Se ele ligar, eu atendo.* — If he calls, I'll answer.
- *Se você estudar, você passa no exame.* — If you study, you'll pass the exam.

The main clause uses presente indicativo (informal future) or futuro indicativo (more formal).

### Exemplar set

- *Se eu tenho tempo, eu cozinho.* — If I have time, I cook. (habitual)
- *Se eu tiver tempo amanhã, eu cozinho.* — If I have time tomorrow, I'll cook. (specific future — `tiver` is futuro do subj of *ter*)
- *Se chove, fico em casa.* — If it rains, I stay home. (general)
- *Se chover, fico em casa.* — If it rains, I'll stay home. (this specific occasion)

## Traps

- **Don't use subjuntivo presente in Type 1.** *Se ele venha, eu ajudo* — wrong (subjuntivo presente). Type 1 takes indicative or futuro do subjuntivo. The hypothetical/wish reading would be Type 2: *Se ele viesse, eu ajudaria*.
- **Pattern 1a vs 1b is a tense-of-the-se-clause distinction.** Habitual takes presente indicativo; specific-future takes futuro do subjuntivo. The forms can look identical for regular verbs (chover → chove vs chover → chover) but differ for irregulars (ter → tem vs ter → tiver).
- **EP overlap question.** EP also uses futuro do subjuntivo for specific-future Type 1, so Pattern 1b is largely shared. Pattern 1a differs less between variants. The EP twin (Cluster 5+) will refine.
- **Type 1 is NOT the place for "se eu fosse rico".** That's Type 2 (irreal). Type 1 is "se eu sou / se eu for", indicative-only.
