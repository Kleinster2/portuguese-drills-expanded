## Outcomes

- Name 21 technology and communication terms
- Use LIGAR correctly (for calling and turning on)
- Use communication verbs (mandar, enviar, receber)
- Talk about technology use and habits
- Ask for WiFi password, phone numbers, etc.

## Vocabulary

- **Devices:**
  - o celular (cell phone)
  - o telefone (telephone)
  - o computador (computer)
  - o notebook (laptop)
  - o tablet (tablet)
  - a televisão / TV (television)
  - o relógio (watch/clock)

- **Communication:**
  - ligar (to call)
  - mandar mensagem (to send message)
  - enviar (to send)
  - receber (to receive)
  - responder (to respond/reply)
  - a mensagem (message)
  - o email (email)
  - o WhatsApp (WhatsApp)

- **Internet:**
  - a internet (internet)
  - o site (website)
  - o WiFi (WiFi)
  - a senha (password)
  - baixar (to download)
  - postar (to post)
  - o app / aplicativo (app)

## Grammar

- **Technology vocabulary often borrowed from English:**
  - WhatsApp, email, WiFi, site, app - same as English!
  - Pronounced with Portuguese accent but spelled similarly
  - Many tech terms are international

- **LIGAR (to call) - Multiple meanings:**
  - **ligar para** + person: "Eu ligo **para** você" (I call you)
  - **ligar** + device: "Eu ligo o computador" (I turn on the computer)
  - Same verb, different meanings based on context!
  - Use PARA when calling a person

- **MANDAR vs ENVIAR (both mean "to send"):**
  - **mandar** is more casual/common: "Eu mando uma mensagem"
  - **enviar** is more formal: "Eu envio um email"
  - Both are correct, mandar is more conversational

- **Using technology verbs:**
  - Eu **uso** o celular. (I use the cell phone.)
  - Eu **ligo** para minha mãe. (I call my mother.)
  - Eu **mando** mensagem. (I send a message.)
  - Eu **baixo** apps. (I download apps.)

### Exemplar sentences

- Eu uso o celular todos os dias. (I use my cell phone every day.)
- Eu ligo para minha família toda semana. (I call my family every week.)
- Eu mando mensagem no WhatsApp. (I send messages on WhatsApp.)
- Você tem WiFi? (Do you have WiFi?)
- Qual é a senha do WiFi? (What's the WiFi password?)
- Eu envio emails para o trabalho. (I send emails for work.)
- Eu baixo muitos apps. (I download many apps.)
