## Outcomes

- Distinguish between the two main translations of "for" in Portuguese
- Express purpose with para + infinitive
- Express duration with por + time expression
- Talk about recipients with para
- Explain reasons with por
- Use fixed expressions like "por favor," "por exemplo," "por isso"
- Understand when to use each preposition correctly

## Vocabulary

- **Prepositions:**
  - **para** (for/to - purpose, destination, recipient, deadline)
  - **por** (for/by/through - reason, duration, agent, means)
- **Vocabulary for common uses:**
  - presente (gift/present)
  - razão (reason)
  - causa (cause)
  - através de (through - alternative to por)
  - durante (during - alternative to por)
  - motivo (reason/motive)
- **Time expressions with por:**
  - por uma hora (for an hour)
  - por dois dias (for two days)
  - por muito tempo (for a long time)
  - por enquanto (for now)
- **Common expressions:**
  - por favor (please - literally: for favor)
  - por exemplo (for example)
  - por que (why - literally: for what)
  - por isso (therefore/that's why - literally: for this)

## Grammar

- **PARA - purpose, destination, recipient, deadline:**
  1. **Purpose/goal** (in order to): Para estudar (in order to study)
  2. **Destination** (to/toward): Vou para casa (I go home/toward home)
  3. **Recipient** (for someone): É para você (It's for you)
  4. **Deadline** (by a time): Para amanhã (by tomorrow)
  5. **Opinion/perspective**: Para mim (for me/in my opinion)

- **POR - reason, duration, agent, means, location:**
  1. **Reason/cause** (because of): Por amor (for love/because of love)
  2. **Duration** (for a length of time): Por duas horas (for two hours)
  3. **Agent** (by someone - passive voice): Escrito por João (written by João)
  4. **Means/method** (by/through): Por email (by email)
  5. **Movement through**: Por aqui (through here)
  6. **Exchange**: Paguei 50 reais por isso (I paid 50 reais for this)

- **Key distinction:**
  - **PARA** = destination, goal, purpose, future-oriented
  - **POR** = reason, cause, duration, past-oriented

### Exemplar sentences

- Estudo para aprender português (I study to learn Portuguese) - purpose
- Isso é para você (This is for you) - recipient
- Vou para o Brasil (I'm going to Brazil) - destination
- Este projeto é para amanhã (This project is for tomorrow) - deadline (with SER)
- Preciso terminar até amanhã (I need to finish by tomorrow) - deadline (with ATÉ, not PARA)
- Trabalho por dinheiro (I work for money) - reason
- Fiz isso por amor (I did this for love) - reason
- Morei lá por três anos (I lived there for three years) - duration
- Mandei por email (I sent by email) - means
- Obrigado por tudo (Thank you for everything) - reason
- Por favor (Please) - fixed expression
