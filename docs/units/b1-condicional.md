## Outcomes

After this unit, the student can:

- Form the **condicional simples** in BP and EP (regular and irregular stems)
- Recognize the parallel between condicional endings and imperfeito do indicativo endings (both share *-ia / -ias / -ia / -íamos / -iam*)
- Use the form productively in any reported-speech, hypothetical, or polite-request context — variant-specific usage is covered downstream

## Vocabulary

- **falar** → falar**ia**, falar**ias**, falar**ia**, falar**íamos**, falar**iam**
- **comer** → comer**ia**, comer**ias**, comer**ia**, comer**íamos**, comer**iam**
- **partir** → partir**ia**, partir**ias**, partir**ia**, partir**íamos**, partir**iam**

## Grammar

The condicional simples is built on the same stems as the synthetic future (`b1-fut-sintetico`), with personal endings shared with the imperfeito do indicativo.

### Form: infinitive + ending

| Pessoa | Ending | Example (falar) |
|---|---|---|
| eu | -ia | falaria |
| tu | -ias | falarias |
| você / ele / ela | -ia | falaria |
| nós | -íamos | falaríamos |
| vocês / eles / elas | -iam | falariam |

The full infinitive serves as the stem (no clipping), with endings appended directly. This is the same pattern as the futuro do indicativo (which uses *-ei / -ás / -á / -emos / -ão*).

### Irregular stems (3 verbs only)

| Verb | Stem | Example (eu form) |
|---|---|---|
| dizer | dir- | diria |
| fazer | far- | faria |
| trazer | trar- | traria |

These three verbs use clipped/altered stems for both the futuro do indicativo and the condicional. All other verbs use the full infinitive as the stem.

### Minimal exemplar set

- *Eu **gostaria** de viajar para Portugal.* — I'd like to travel to Portugal.
- *Tu **comerias** mais se tivesses fome?* — Would you eat more if you were hungry?
- *Ele **diria** a verdade se pudesse.* — He'd tell the truth if he could.
- *Nós **viajaríamos** mais cedo, mas não dá.* — We'd travel earlier, but it's not possible.
- *Eles **fariam** isso por nós.* — They'd do that for us.

## Traps

- **Don't confuse the endings with imperfeito do indicativo.** Condicional: *falaria*. Imperfeito: *falava*. Both share the *-ia/-ias/-ia/-íamos/-iam* PERSONAL endings, but the stem differs. Imperfeito clips the infinitive (*fal-ava*); condicional appends to the full infinitive (*falar-ia*).
- **Only 3 irregular stems.** *Dizer → diria, fazer → faria, trazer → traria*. Every other verb is regular (*ser → seria, ter → teria, vir → viria, ir → iria, pôr → poria, estar → estaria*).
- **This unit teaches form, not usage.** When to *use* condicional — for hypothetical Type 2 main clauses, for politeness, for reported-speech future-backshift — is variant-conditional and covered in `b1-condicional-bp` / `b1-condicional-ep` and downstream cluster units.
- **Spoken EP often substitutes imperfeito do indicativo for condicional in informal speech** (*Eu queria* vs *Eu quereria*). That's a usage choice, not a form gap; the form *quereria* exists and conjugates as expected. The substitution is documented in [[b1-condicional-ep]].
