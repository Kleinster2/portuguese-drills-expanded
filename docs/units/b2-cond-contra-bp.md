## Outcomes

After this unit, the student can:

- Construct **Type 3 (counterfactual past)** conditional sentences in BP using `se + mais-que-perfeito subjuntivo + condicional composto`
- Recognize Type 3 as the past-time counterpart of Type 2 — the imagined situation is no longer reachable because the moment is gone
- Distinguish *Se eu tivesse estudado, teria passado* (counterfactual past) from *Se eu estudasse, passaria* (irreal present/future)

## Vocabulary

- **se** — if (counterfactual past reading)
- *mais-que-perfeito subjuntivo* forms (review from `b2-mais-que-perfeito-subj`): tivesse + particípio
- *condicional composto* forms (review from `b2-condicional-composto-bp`): teria + particípio

## Grammar

Type 3 covers situations the speaker treats as **counterfactual** — what *would have* happened *if* something *had been* different in the past. The past time anchor closes the door; the counterfactual is no longer reachable.

Pattern (rigid in BP):

`se + [mais-que-perfeito subjuntivo: tivesse + particípio] + [condicional composto: teria + particípio]`

### Exemplar sentences

- *Se eu **tivesse estudado** mais, **teria passado** no exame.* — If I had studied more, I would have passed the exam.
- *Se ela **tivesse vindo** ontem, **teríamos comemorado** juntos.* — If she had come yesterday, we would have celebrated together.
- *Se vocês **tivessem chegado** mais cedo, **teriam visto** o início do filme.* — If you had arrived earlier, you would have seen the beginning of the movie.
- *Se eu **tivesse sabido**, **teria avisado** todo mundo.* — If I had known, I would have warned everyone.
- *Se ele **tivesse dito** a verdade, ninguém **teria ficado** chateado.* — If he had told the truth, nobody would have gotten upset.

### Type 2 vs Type 3 contrast

The difference is the time anchor of the imagined situation:

- *Se eu **estudasse**, **passaria**.* — If I studied (now / in general), I'd pass. (Type 2 — present/future hypothetical)
- *Se eu **tivesse estudado**, **teria passado**.* — If I had studied (back then), I would have passed. (Type 3 — counterfactual past)

Both are unreal at the moment of speaking; Type 3 specifically anchors the unreality in the past.

### Mixed conditionals

A common variant: past condition → present consequence (mixing Type 3's *se*-clause with Type 2's main clause):

- *Se eu **tivesse estudado**, **estaria** trabalhando como engenheiro hoje.* — If I had studied, I'd be working as an engineer today.

The *se*-clause is mais-que-perfeito subjuntivo (past condition); the main clause shifts to condicional simples (present consequence) because the result is in the present.

## Traps

- **Don't substitute mais-que-perfeito do indicativo (tinha estudado) for the subjuntivo form (tivesse estudado).** *Se eu tinha estudado, teria passado* — wrong in BP. The *se*-clause requires subjuntivo. (EP often allows *tinha estudado* in spoken usage, which is the source of EP-twin divergence — covered in the EP twin.)
- **Don't use the simple condicional in the main clause for past consequences.** *Se eu tivesse estudado, passaria* mixes Type 3's *se*-clause with Type 2's main clause — only acceptable when the consequence is genuinely present (mixed conditional, see above). For straightforward past consequence, use *teria passado*.
- **EP register substitution.** EP commonly says *Se eu tivesse estudado, tinha passado* (mais-que-perfeito do indicativo replacing condicional composto). BP keeps *teria passado*. Recognize the EP form when you hear it.
