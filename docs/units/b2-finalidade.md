## Outcomes

After this unit, the student can:

- Construct purpose clauses ("in order to / so that") using **para que / a fim de que + subjuntivo**
- Choose the **personal-infinitive alternative** (*para + infinitive*) when the subject of the purpose clause is the same as the main clause OR when distinct (with personal-infinitive inflection)
- Distinguish purpose clauses from result clauses (consecutivas) and concessive clauses (concessivas)

## Vocabulary

The purpose-conjunction inventory:

| Conjunction / structure | Mood / form | Register |
|---|---|---|
| **para que** + subj | subjuntivo | most common |
| **a fim de que** + subj | subjuntivo | formal |
| **a fim de** + infinitivo | infinitivo | formal, less common |
| **para** + infinitivo | bare or personal infinitive | most natural in BP/EP |
| **com a finalidade de** + infinitivo | infinitivo | very formal |
| **no intuito de** + infinitivo | infinitivo | formal/literary |

## Grammar

Purpose clauses express the GOAL of the main-clause action — what the action aims to bring about. Two structurally distinct paths exist:

1. **Conjunction + subjuntivo** (clause structure)
2. **Para + infinitivo** (cleaner, more common in spoken usage)

The choice between them is partly subject-determined and partly register-determined.

### Same subject — bare infinitive is the natural choice

When the subject of the main clause and the purpose clause is **identical**, the natural form is *para + bare infinitive*. Adding *que* + subjuntivo with a redundant subject sounds heavy:

- *Estudo muito **para passar** no exame.* — I study a lot to pass the exam. (natural)
- *~~Estudo muito **para que eu passe** no exame.~~* — Same meaning, but the *eu* is redundant; sounds bookish.

### Distinct subject — choose between *para que* + subj and *para* + personal infinitive

When the purpose clause has a **different subject** from the main clause, both options exist and are productive:

- *Estudo muito **para que os meus pais fiquem** orgulhosos.* — I study so my parents are proud. (subjuntivo)
- *Estudo muito **para os meus pais ficarem** orgulhosos.* — Same meaning. (personal infinitive)

The personal-infinitive form is **more economical and slightly more formal in writing**; the subjuntivo form is **more emphatic and slightly more conversational**. Both are correct.

See [[b2-infinitivo-pessoal-uso]] for the personal-infinitive details.

### Formal alternatives

For deliberately formal register (academic, legal, official) the longer conjunctions appear:

- *Trabalhamos **a fim de que** todos **tenham** acesso.* — We work so that everyone has access. (formal, with subjuntivo)
- *Trabalhamos **com a finalidade de proporcionar** acesso a todos.* — We work in order to provide access to everyone. (formal, with infinitivo)
- *Reformamos a lei **no intuito de proteger** os consumidores.* — We reformed the law in order to protect consumers. (formal/literary)

These are recognition targets in formal text; for production at B2, *para que* + subj and *para* + (personal) infinitivo cover almost all needs.

### Tense in the subjuntivo subordinate

The purpose-clause subjuntivo follows the main verb's tense:

- Main verb in present/future → presente subjuntivo: *Trabalho **para que ele tenha** uma vida melhor.*
- Main verb in past → imperfeito subjuntivo: *Trabalhei **para que ele tivesse** uma vida melhor.*

### Exemplar set

- *Saí cedo **para chegar** a tempo.* — I left early to arrive on time. (same subject, bare infinitive)
- *Saí cedo **para o João chegar** a tempo.* — I left early so João would arrive on time. (distinct subject, personal infinitive)
- *Saí cedo **para que o João chegasse** a tempo.* — Same meaning, subjuntivo path.
- *A professora explicou devagar **para que todos percebessem**.* — The teacher explained slowly so everyone would understand.
- *Estudamos juntos **a fim de termos melhores resultados**.* — We study together so we have better results. (formal, personal infinitive)
- *O governo aprovou a lei **no intuito de proteger** as crianças.* — The government passed the law to protect the children. (formal/literary)

## Traps

- **Don't use *para que* with the same subject.** *Estudo para que eu passe* sounds wrong/bookish. Same subject → *para passar*.
- **The subjuntivo path requires a *que*.** *Estudo para os meus pais fiquem* — wrong (no *que*). Either *para que os meus pais fiquem* (with *que* and subj) or *para os meus pais ficarem* (no *que*, personal infinitive).
- **A fim de vs a fim de que.** *A fim de* takes infinitive (often with same subject); *a fim de que* takes subjuntivo. Both are formal; the difference is the same as *para* vs *para que*.
- **Don't confuse with consecutive clauses.** Purpose: *Falo devagar **para que percebas**.* (subjuntivo, intent). Result: *Falo devagar **de modo que percebes**.* (indicative, consequence). The conjunction *de modo que* is genuinely ambiguous between purpose and result depending on the verb mood — see [[b2-consecutivas]] for the result-clause framing.
- **Tense agreement with main clause.** Past main verb forces past subjuntivo: *Estudei para que ele **passasse***. — not *para que ele passe*. The subordinate aligns to the temporal frame of the main clause.
- **Don't substitute *para* with *por* in purpose clauses.** *Por* expresses cause/exchange/duration, not purpose. *Estudo por passar no exame* is wrong — use *para passar*.
