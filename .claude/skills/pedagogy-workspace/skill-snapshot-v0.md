# Load Pedagogy

Read and internalize the pedagogical methodology from PEDAGOGY.md file in the project root. This document contains all core principles for Portuguese drill design including:

- Self-introduction first ("eu" before all others)
- Yes/No verb echo pattern (verb shifts person in answer)
- Conjugation-pattern pedagogy (singular before plural)
- Grammar in service of communication
- Centrality ranking for vocabulary (all vocab, not just verbs)
- Vocabulary scaffolding (define before use, never repeat)
- Pronunciation examples preview later vocabulary
- The Self-Introduction Trinity (morar, falar, trabalhar)
- Native Usage Filter (never teach a form nobody uses)
- BP/EP variant separation

After reading PEDAGOGY.md, confirm understanding of the key principles and be ready to apply them.
