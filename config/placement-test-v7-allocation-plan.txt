================================================================================
PORTUGUESE PLACEMENT TEST V7.0 - PRAGMATIC ALLOCATION PLAN
================================================================================

PHILOSOPHY CHANGE:
Old: Rigid 90 units × 2 questions (1C + 1P) = 180 questions
New: Allocate questions based on WHAT NEEDS TESTING, not arbitrary rules

Target: ~200-220 strategic questions that test meaningful competencies


================================================================================
UNIT CATEGORIZATION & ALLOCATION STRATEGY
================================================================================

CATEGORY 1: PURE GRAMMAR UNITS (Contractions, Articles, Agreement)
   Allocation: 0-1 comprehension + 3-5 production
   Rationale: Must test ALL variations of grammar rules, not just one example

CATEGORY 2: VOCABULARY/MEANING UNITS (Question words, Prepositions, Time)
   Allocation: 3-5 comprehension + 0-1 production
   Rationale: Multiple vocabulary items per unit need individual testing

CATEGORY 3: BALANCED VERB UNITS (Basic conjugations, Simple statements)
   Allocation: 1 comprehension + 1-2 production
   Rationale: Test understanding + basic conjugation ability

CATEGORY 4: COMPLEX TENSE/MOOD UNITS (Subjunctive, Conditional, Preterite)
   Allocation: 1-2 comprehension + 2-4 production
   Rationale: Complex conjugation patterns need more production questions

CATEGORY 5: RECOGNITION-ONLY UNITS (Negation, Word order, Usage distinctions)
   Allocation: 1-2 comprehension + 0 production
   Rationale: Just need to recognize/understand, not actively produce

CATEGORY 6: USAGE/NUANCE UNITS (Ser vs estar, Por vs para, Ficar usage)
   Allocation: 2-3 comprehension + 1-2 production
   Rationale: Test understanding of WHEN to use which option
